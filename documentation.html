    Formularios

    Los formularios basicamente cogen el estilo estandar de cada navegador tal y como estan en el estilo actual

    Label:

    			<label>Label</label>

    Input:

    Para introducir un input con su correspondiente label se haria de la siguiente forma

    			<label>Nombre:</label><input>

    Si se quiere posicionar el input debajo se añade al label la clase clearfix

    			<label class="clearfix">Nombre:</label><input>

    Select:

    	    		<label>Selecciona un Producto:</label>
			<select name="" id="">Selecciona un Producto
				<option value="">1</option>
				<option value="">2</option>
				<option value="">3</option>
			</select>

    Textarea:

    Tambien se pueden crear elementos con la clase w* para establecer un ancho. (Ejemplo: la clase w500 establece un ancho de 500px.)

    		    		<label>Introduce tu mensaje:</label>
				<textarea id="" class="w500"></textarea>

    Abstrats

    Existen clases estandar que nos ayudan a dar estilo a el documento.

    Position

    .relative Posicion relativa.

    .absolute Posicion absoluta.

    .clearfix Hace un clear en el elemento.

    .left Float left.

    .right Float right.

    .t(n) top: (n)px Ejemplo: .t5 => top:5px.

    .b(n) bottom: (n)px Ejemplo: .b5 => bottom:5px.

    .l(n) left: (n)px Ejemplo: .l5 => left:5px.

    .r(n) right: (n)px Ejemplo: .r5 => right:5px.


    Layout

    .hide,.hidden Display none.

    .visible Display block.


    Aligns

    .align_center Text align center.

    .align_right Text align right.

    .align_left Text align left.

    .align_justify Text align justify.


    Elastics widths

    .e(n) width: (n)% Ejemplo: .e30 => width:30%;


    Fixed widths

    .w(n) width: (n)px Ejemplo: .w100 => width:100px;


    Fixed heigths

    .h(n) height: (n)px Ejemplo: .h100 => height:100px;


    Line heigths

    .lh(n) line-height: (n)px Ejemplo: .lh50 => line-height:5px;


    Margins

    .m(n) margin: (n)px Ejemplo: .m50 => margin:50px;

    .mt(n) margin-top: (n)px Ejemplo: .mt50 => margin-top:50px;

    .mr(n) margin-right: (n)px Ejemplo: .mr50 => margin-right:50px;

    .mb(n) margin-bottom: (n)px Ejemplo: .mb50 => margin-bottom:50px;

    .ml(n) margin-left: (n)px Ejemplo: .ml50 => margin-left:50px;


    Paddings

    .p(n) padding: (n)px Ejemplo: .p50 => padding:50px;

    .pt(n) padding-top: (n)px Ejemplo: .pt50 => padding-top:50px;

    .pr(n) padding-right: (n)px Ejemplo: .pr50 => padding-right:50px;

    .pb(n) padding-bottom: (n)px Ejemplo: .pb50 => padding-bottom:50px;

    .pl(n) padding-left: (n)px Ejemplo: .pl50 => padding-left:50px;


    Opacity

    .opacity_(n) opacity: 0.n Ejemplo: opacity_5 => opacity: 0.5.


    Colors

    .color-blue* color: blue. (*blue, green, red, pink, purple, white)

    .blue* background-color: blue. (*blue, green, red, pink, purple, white)
